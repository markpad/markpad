# Como organizar conteudo em papel pautado

## Introducao

Esse layout funciona melhor quando o texto acompanha exatamente a distancia entre as linhas azuis do fundo.

Em vez de confiar nas margens padrao do navegador, voce define cada bloco com uma altura de linha previsivel.

## Boas praticas

Use leading fixo, remova margens padrao e insira espacadores em multiplos da pauta.

- Defina `leading-[40px]` para paragrafos e itens de lista.
- Use titulos com o mesmo ritmo vertical para manter consistencia.
- Reserve espacos maiores apenas entre secoes.

> Quando o baseline e consistente, a leitura fica mais organizada e natural.

### Checklist rapido

1. Ajustar `body` com gradiente de caderno.
2. Remover margens padrao dos elementos.
3. Validar o alinhamento no preview e no export.
