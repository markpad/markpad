# How to organize content on ruled paper

## Introduction

This layout works best when text follows the exact spacing of the blue ruled lines in the background.

Instead of relying on default browser margins, you define each block with a predictable line height.

## Best practices

Use fixed leading, remove default margins, and insert spacing in multiples of the ruled grid.

- Set `leading-[40px]` for paragraphs and list items.
- Keep headings on the same vertical rhythm for consistency.
- Use larger spacing only between sections.

> When the baseline is consistent, reading feels more organized and natural.

### Quick checklist

1. Configure `body` with the notebook gradient.
2. Remove default element margins.
3. Validate alignment in preview and export.
