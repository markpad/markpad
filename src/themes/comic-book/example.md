## Origin Story

Every hero needs an origin story. This theme draws inspiration from the **golden age of comics**, bringing:

- **Bold typography** that pops off the page
- **Vibrant colors** inspired by classic four-color printing
- **Dynamic shadows** that create depth and impact
- **Hand-drawn energy** with wavy decorations

## The Code of Heroes

```javascript
function assembleTeam(heroes) {
  return heroes
    .filter((hero) => hero.isAvailable)
    .map((hero) => ({
      ...hero,
      status: 'ASSEMBLED! 💪',
    }))
}

const avengers = assembleTeam(allHeroes)
console.log('Heroes assembled:', avengers.length)
```
