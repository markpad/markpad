# Accessible Reading

A theme designed for everyone, with extra attention to readability.

## Why Accessibility Matters

Reading should be comfortable for **everyone**. This theme uses larger spacing, clear fonts, and high contrast to ensure the best possible reading experience.

### Key Design Choices

1. Generous line height and letter spacing
2. Sans-serif font for screen readability
3. Warm, low-glare background color
4. High contrast without harsh black-on-white

> "The power of the Web is in its universality. Access by everyone regardless of disability is an essential aspect."
> — Tim Berners-Lee

## Readability Guidelines

| Feature        | This Theme | Standard  |
| -------------- | ---------- | --------- |
| Line Height    | 2.0        | 1.5       |
| Letter Spacing | Wide       | Normal    |
| Font Size      | 18px base  | 16px base |
| Paragraph Gap  | 1.5rem     | 1rem      |

## Text Example

```
This is a sample of monospace text.
```
