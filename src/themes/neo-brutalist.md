# BRUTALISM IN DESIGN

**NO APOLOGIES. NO COMPROMISES.**

## WHAT IS NEO-BRUTALISM?

Neo-brutalism rejects the polished, over-designed aesthetic of modern web design. It embraces **raw edges**, **bold colors**, and **unapologetic simplicity**.

### KEY PRINCIPLES

1. THICK BORDERS
2. HIGH CONTRAST
3. PRIMARY COLORS
4. NO GRADIENTS
5. BOLD SHADOWS

> "DESIGN SHOULD NOT BE INVISIBLE. IT SHOULD PUNCH YOU IN THE FACE."

## THE COLOR PALETTE

| COLOR  | HEX       | USE           |
| ------ | --------- | ------------- |
| BLACK  | `#000000` | BORDERS, TEXT |
| YELLOW | `#FACC15` | ACCENT        |
| BLUE   | `#2563EB` | LINKS         |
| WHITE  | `#FFFFFF` | BACKGROUND    |

### CODE BLOCK

```css
.brutalist {
  border: 4px solid black;
  box-shadow: 8px 8px 0 black;
  background: white;
}
```
